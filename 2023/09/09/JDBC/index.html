JDBC(Java DataBase Connectivity)(Java数据库连接)是java语言操作关系型数据库的一套API

JDBC API 详解

DriverManager

  • 作用
    • 注册驱动
    • 获取数据库连接

Connection

  • 作用

    1. 获取执行SQL的对象

      • 普通执行SQL对象

      • 预编译SQL的执行SQL对象：防止SQL注入

      • 执行存储过程的对象

    2. 管理事务

      • MySQL事务管理

      • JDBC事务管理
